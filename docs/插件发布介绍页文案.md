# MindMark

把 Obsidian 笔记变成可以编辑的思维导图。

MindMark 是一款为 Obsidian 设计的思维导图插件。它不会要求你维护一套新的文件格式，也不需要在笔记和脑图之间来回复制内容。你仍然写 Markdown，只是在需要整理结构、梳理关系、展开想法的时候，可以把同一篇笔记切换成思维导图来查看和编辑。

## 项目来源与致谢

MindMark 基于 [markmindckm/obsidian-enhancing-mindmap](https://github.com/markmindckm/obsidian-enhancing-mindmap) 修改而来。

原项目为 Obsidian 中的 Markdown 思维导图体验打下了很好的基础，在此向原作者表示感谢。MindMark 的修改主要是为了让这类工作流在新的 Obsidian 使用环境中继续可用，并补上一些日常使用中容易遇到的体验缺口。

由于原项目已经较长时间没有维护，部分功能在当前 Obsidian 中会遇到兼容性问题，也有一些操作逻辑不够完整。MindMark 在此基础上做了维护和调整，重点放在视图切换、节点编辑、双链联动、导出、样式细节和日常可用性上。

## Markdown 还是 Markdown，只是多了一种看法

很多时候，我们用 Markdown 写大纲、读书笔记、项目拆解或知识卡片。文字很适合记录细节，但当内容变多之后，层级关系很容易藏在一长串标题和列表里。

MindMark 做的事情很简单: 让这些标题和列表变成一张可以操作的脑图。

你可以在 Markdown 里继续写，也可以切到脑图里拖动、展开、折叠和编辑。内容最终仍然保存回原来的 Markdown 文件中。

## 适合用来整理结构化内容

MindMark 特别适合这些场景:

- 拆解一个主题、课程、产品方案或写作提纲。
- 把读书笔记整理成更清楚的知识结构。
- 梳理项目计划、会议内容和行动项。
- 在一大段 Markdown 大纲中快速找到重点分支。
- 把 Obsidian 里的已有笔记换一种方式浏览。

如果你经常在 Obsidian 里写标题、列表和层级结构，MindMark 会很自然地接上你的工作流。

## 在脑图里直接编辑

MindMark 的脑图不是只读预览。你可以直接在节点上编辑内容，新增子节点和兄弟节点，删除、移动、拖拽整理结构，也可以撤销和重做。

常用操作都尽量贴近键盘习惯:

- `Tab` 新增子节点。
- `Enter` 新增同级节点。
- `Space` 或双击编辑节点。
- 方向键在节点之间移动。
- 拖拽节点调整层级和顺序。

这让它更像是在整理一份活的大纲，而不是导出一张静态图片。

## 保留 Obsidian 的笔记习惯

MindMark 不把你从 Obsidian 的体系里拉出去。

节点里可以使用常见 Markdown 样式，比如加粗、斜体、高亮、删除线、行内代码和链接。Obsidian 内部链接也可以显示在节点中，你可以从脑图里打开相关笔记，也可以使用链接预览来快速查看内容。

对已经习惯双链笔记的人来说，这一点很重要: 脑图只是新的视角，笔记关系仍然留在原来的知识库里。

## 大图也能聚焦查看

当一张脑图越来越大时，MindMark 提供了 Focus Mode。你可以只聚焦当前节点所在的分支，把其他部分淡化掉。

这很适合用在复杂主题里: 先看全局，再临时收窄到某一个分支，避免被整张图的信息量淹没。

## 支持导入和导出

MindMark 支持拖拽导入 `.xmind` 文件，方便把已有脑图内容带进 Obsidian。

需要分享或存档时，也可以导出为:

- PNG 图片
- JPEG 图片
- HTML 文件

无论是发给别人看、放进文档里，还是作为阶段性整理结果保存下来，都比较方便。

## 可以按自己的习惯调整

你可以在设置里调整画布大小、背景色、节点字号、布局方向、分支颜色、移动节点后的自动居中行为，以及 Focus Mode 的遮罩透明度。

内部链接的打开方式也可以选择: 当前标签页、新标签页或新窗口。

## 如何开始

安装并启用 MindMark 后，你可以创建一篇新的思维导图笔记，也可以把符合结构的 Markdown 笔记打开为脑图。

一个简单的笔记结构就可以变成脑图:

```markdown
# 我的主题

## 第一部分
- 要点一
- 要点二

## 第二部分
- 想法 A
- 想法 B
```

你可以先从一篇已有的大纲笔记试起。切换到脑图后，再用拖拽和快捷键整理结构。

## 需要知道的边界

MindMark 更适合整理 Markdown 大纲和结构化笔记，不是完整替代专业脑图软件。

它的优势在于和 Obsidian 笔记自然配合: 文件还是 Markdown，链接还在知识库里，内容可以在文本和脑图之间来回切换。

如果你想要的是复杂样式、演示模板或专业制图能力，专门的脑图软件会更合适。如果你想在 Obsidian 里把笔记结构看清楚、改得更顺手，MindMark 正是为这个场景准备的。

## 一句话总结

MindMark 让 Obsidian 的 Markdown 笔记多了一种可视化编辑方式: 写作时用文本，整理时用脑图，保存时仍然回到你的笔记库。
