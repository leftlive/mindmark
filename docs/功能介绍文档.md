# MindMark 功能介绍文档

## 产品定位

MindMark 是一个基于 Markdown 的 Obsidian 思维导图插件。它的核心思路不是引入独立文件格式，而是让用户直接用 Markdown 文件作为思维导图的数据源，在 Markdown 视图和 MindMap 视图之间切换、编辑和保存。

## 当前已实现的功能

## 1. 文件与视图管理

- 支持创建新的思维导图文件。
- 新文件本质上仍是 Markdown 文件，并写入思维导图 frontmatter 标记。
- 带有指定 frontmatter 的 Markdown 文件可自动以 MindMap 视图打开。
- 支持在当前文件的 Markdown 视图与 MindMap 视图之间切换。
- 支持从 Markdown 视图标题区显示“Open as mindmap board”按钮。
- 支持在文件右键菜单中打开为思维导图。
- 支持在 MindMap 视图中一键切回 Markdown。
- 支持在右侧新建一个 Markdown 分屏查看同一文件。

## 2. Markdown 与思维导图双向转换

- 从 Markdown 解析生成树形节点结构。
- 将思维导图节点树重新序列化回 Markdown。
- 保留 frontmatter，并在保存时把导图内容写回原文件。
- 支持通过标题层级设置决定哪些节点导出为 Markdown 标题。
- 超出标题层级的节点导出为列表项。
- 折叠节点会在导出的 Markdown 中写入块引用式折叠标记 `^id`，用于再次解析时恢复折叠状态。

## 3. 节点显示与渲染

- 使用中心画布展示树状思维导图。
- 支持根节点、二级节点、普通节点的不同视觉层级。
- 支持连接线绘制和颜色主题。
- 支持左右展开和顺时针布局。
- 支持节点内容按 Markdown 渲染，而不是纯文本。
- 支持代码块、强调、行内样式等常见 Markdown 内容显示。
- 支持节点内容中的图片嵌入渲染。
- 支持节点内容中的 Markdown 嵌入内容预览。

## 4. 节点编辑能力

- 双击节点进入编辑。
- 空格键进入编辑。
- 支持编辑结束后自动回写节点文本。
- 新建空导图时自动聚焦根节点并进入编辑状态。
- 支持插入兄弟节点。
- 支持插入子节点。
- 支持删除节点及其子树。
- 支持复制节点子树。
- 支持剪切节点子树。
- 支持粘贴节点子树。
- 支持撤销。
- 支持重做。
- 支持恢复为上一次文本。

## 5. 文本格式化能力

- 支持选中节点文本。
- 支持为节点文本加粗。
- 支持为节点文本斜体。
- 支持为节点文本高亮。
- 支持为节点文本删除线。
- 支持加入缩进。
- 支持插入 `<br>` 换行。
- 支持移除 `<br>` 换行。
- 支持取消编辑。

## 6. 节点结构操作

- 支持展开一层。
- 支持按当前最大可见层级继续展开一层。
- 支持折叠一层。
- 支持按当前最大可见层级回退折叠一层。
- 支持切换节点展开/折叠状态。
- 支持将当前节点上移、下移。
- 支持将当前节点左移、右移。
- 支持把后续兄弟节点收编为当前节点的子节点。
- 支持把所有兄弟节点收编为当前节点的子节点。
- 支持与下一个节点合并。
- 支持按“引用”形式与下一个节点合并。

## 7. 交互与导航

- 支持键盘方向键在节点之间导航。
- 支持 Home 快速定位根节点。
- 支持中心定位当前节点。
- 支持整个导图居中。
- 支持缩放放大。
- 支持缩放缩小。
- 支持鼠标拖动画布。
- 支持拖拽节点改变层级和顺序。
- 拖拽时会显示插入指示线或子节点高亮。

## 8. Focus Mode

- 支持对当前节点进入 Focus Mode。
- Focus Mode 下仅突出当前节点子树。
- 支持退出 Focus Mode。
- 支持配置 Focus Mode 遮罩透明度。

## 9. 双链与文件联动

- 支持节点中 Obsidian 内部链接渲染。
- 支持点击内部链接时打开目标笔记。
- 支持配置内部链接打开方式：
  - 当前标签页
  - 新标签页
  - 新窗口
- 支持悬停内部链接时触发 Obsidian 页内预览。
- 支持在编辑节点时输入 `[[` 后进行文件联想补全。

## 10. 导入与导出

- 支持拖拽导入 `.xmind` 文件。
- 支持从 XMind 内容转换为当前思维导图结构。
- 支持导出 PNG。
- 支持导出 JPEG。
- 支持普通、低清和高清倍率导出。
- 支持导出 HTML。

## 11. 设置项

- 画布尺寸设置。
- 画布背景色设置。
- Markdown 标题层级设置。
- 节点字号设置。
- 布局方向设置。
- 分支颜色预设设置。
- 移动节点后自动居中设置。
- Focus Mode 遮罩透明度设置。
- 双链悬停预览开关。
- 双链点击打开方式设置。

## 12. 国际化

- 已包含多语言文案资源。
- 至少可见中英文文案支持，并预置多个语种 locale 文件。

## 当前功能边界

- 数据文件仍然是 Markdown，不是独立的二进制思维导图格式。
- XMind 目前是导入能力，不是完整双向同步。
- “导出 HTML” 从实现看更接近导出图片嵌入片段，不是完整的独立网页。

## 适合的使用场景

- 用 Markdown 维护知识结构，同时需要图形化浏览。
- 在 Obsidian 中进行读书笔记、论文摘录、主题拆解和大纲整理。
- 希望在文本编辑和脑图浏览之间快速切换，而不维护两套数据。
